## Sobre o projeto
O objetivo do trabalho é construir modelos de Machine Learning para prever qual time vencerá a rodada (Contra-Terroristas ou Terroristas) em partidas de CS:GO. Para isso, são utilizados dados em tempo real da partida, como vida dos jogadores, mapa atual, dinheiro acumulado, armas e utilitários comprados.

## Conteúdo do repositório
* **Somativa1.ipynb** (e versão .html): Análise exploratória dos dados (EDA), pré-processamento inicial e testes com algoritmos.
* **Somativa2.ipynb** (e versões .html e .pdf): Construção de pipelines de pré-processamento e modelagem usando Scikit-Learn, seleção de atributos, otimização de hiperparâmetros (GridSearchCV e RandomizedSearchCV) e avaliação de métricas dos modelos (como Random Forest).

## Requisitos para rodar

pip install pandas numpy matplotlib seaborn scikit-learn ydata-profiling notebook
